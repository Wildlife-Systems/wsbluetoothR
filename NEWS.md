# wsbluetoothtracker 0.1.0

## Initial Release

* High-performance Bluetooth data processing with Rcpp
* `process_bluetooth_fast()` function for processing large datasets
* Summary and visualization functions for device detection analysis
* Power-weighted centroid calculations
* Cross-platform support (Windows, Linux, macOS)

## Functions

* `process_bluetooth_fast()`: Process Bluetooth RSSI data files
* `summarize_device_counts()`: Generate device statistics
* `plot_device_counts()`: Visualize detection counts over time
* `compare_devices()`: Multi-device comparison plots
* `get_device_timeseries()`: Extract time-series for specific devices
